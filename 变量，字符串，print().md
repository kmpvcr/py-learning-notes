#### 一、梦开始的地方

下面我们用python写一个猜数字的游戏

```python
""" 用python设计的第一个小游戏 """
import random

guess_number = random.randint(1,10)
counts = 3
while counts > 0:
    temp = input("猜数字：")
    guess = int(temp)

    if guess == guess_number:
        print("猜中了")
        break
    elif guess < guess_number:
        print("猜小了")

    else:
        print("猜大了")

    counts -=1
        
print("游戏结束")
```



#### 二、使用变量

```python
my_str = '字符串'

```

```python
my_str = '字' + '符串'

```

```python
a = '这是一句话的前半段，'
b = '这是上一句话的后半段'
c = '前半段加后半段话是一段完整的话'
sentence = a + b + c
```

我们经常将一个值赋给变量，然后随时使用它。

---

命名变量值得注意的规则：

- 变量名只能包含字母、数字和下划线。
- 变量名能以字母和下划线打头，但不能以数字打头
- 不要将python关键字和函数名用作变量名

命名变量的建议：

**变量 / 函数**：小写 + 下划线

**类名**：大驼峰（每个单词首字母大写）



#### 三、字符串的说明

三种基础字符串：

```python
'单引号字符串'

"双引号字符串"

'''我喜欢
阳光，
大地，
河流，
树林，
空气'''        #三重引号支持跨行字符串，一般用于文档字符串

```

转义字符的使用：

```python
"Let's go!"   #1.双引号中使用单引号。不能在单引号字符串中再次使用单引号，否则引号匹配错误；双引号字符串同理。

'Let\'s go!'  #2.转义字符特殊处理后面的字符

'C:\\'        #3.反斜杠对自身转义，以便取消转义

print(r'C:\' )  #4.打印r字符串时输出原始字符
```





#### 四、print()使用说明

```python
print("I LOVE CHINA")

print(1314520)

print("I love you" * 3000)
```

输出的内容都在一行。

```python
print("I love you\n" * 3000)
```

此时出现换行。

---

如果希望多行输入，可以在一行末尾加上`\`

例如：

```python
print("这条语句太长了，一行写不下，\
所以我选择加上'\\'，就能换行继续写。")
```

当然输出的结果是 `这条语句太长了，一行写不下，所以我选择加上'\'，就能换行继续。`，相比于三重引号，引号内容中的换行是不会输出的，最终的结果仍旧处在一行，除非每次手动换行时加上`\n`
