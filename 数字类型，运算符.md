#### 一、python内置的数字类型

1.`int`型

```python
a = 0

b = int("0")
```

2.`float`型

```python
c = 0.0

d = float("0.0")
```

---

四则运算的结果很多出乎人的意料，比如`0.1 + 0.2`的输出结果是`0.30000000000000004`，从而python认为`0.1 + 0.2`等于`0.3`是假的。这是计算机能表示的二进制进度有限导致的错误。采用decimal模块中的Decimal类可以避免此类问题出现。

```python
import decimal
a = decimal.Decimal('0.1')
b = decimal.Decimal('0.2')
print(a + b)
```



3.`bool`型

```python
e = True
f = False
g = bool('')
```



哪些值被认为是`False`呢？

```python
定义为False的对象：None 和 False
值为0的数字类型：0，0.0，0j，Decimal(0),Fraction(0,1)
空的序列和集合：'',(),[],{},set(),range()
```
